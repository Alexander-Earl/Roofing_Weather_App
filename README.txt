Mac/Linux Instructions:
1. Open a terminal in this directory
2. Run 'npm install' to install the dependencies
3. Run 'npm run dev' to run the web application
4. Open a browser to http://localhost:8080

Windows Instructions:
1. Open Powershell or Command Prompt in this directory
2. Run 'npm install' to install the dependencies
3. Run 'npm run dev' to run the web application
4. Open a browser to http://localhost:8080

NOTE: Please ensure that the web address has localhost and not your computer's IP address of 127.0.0.1. This is because your browser will block the Geolocation
API as it can only run on secure connections or localhost. If geolocation is blocked this means that the app cannot request GPS access.
